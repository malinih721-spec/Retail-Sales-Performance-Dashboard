# Retail Sales Performance Dashboard using Power BI

## Overview
This project presents an interactive Power BI dashboard designed to analyze retail sales performance and provide business insights through data visualization.

## Features
- Sales Performance Analysis
- Revenue Trend Analysis
- Product Category Insights
- Regional Sales Analysis
- Key Performance Indicator (KPI) Dashboard

## Tools Used
- Power BI
- Microsoft Excel
